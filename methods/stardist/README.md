# StarDist

## Overview

This folder contains StarDist-related scripts, environment files, and workflow notes used in the segmentation analysis.

## Workflow

StarDist was included as one of the image-based segmentation methods in the benchmark workflow.

Pretrained models were selected according to the input image type.

## Environment Files

The environment file in this folder records the software dependencies used for running StarDist.

## Scripts

Running scripts are provided for the StarDist workflow.

Input and output paths should be updated according to the local computing environment.

## Parameter Notes

StarDist was mainly run with default or standard settings to keep the analysis comparable across methods.

Large image files and complete segmentation outputs are not stored in this repository.

## Reference

Official StarDist repository: https://github.com/stardist/stardist
