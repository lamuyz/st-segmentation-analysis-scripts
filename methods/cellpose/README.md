# Cellpose

## English

This folder contains Cellpose-related scripts, environment files, and workflow notes used in the segmentation analysis.

### Included Models

| Version | Model |
|---|---|
| Cellpose2 | `cyto2` |
| Cellpose3 | `cyto3` |
| Cellpose3 | `nuclei` |
| Cellpose4 | `cpsam` |

### Environment Files

Three environment files are included for different Cellpose versions:

- Cellpose2 environment
- Cellpose3 environment
- Cellpose4 environment

These files record the software dependencies used during analysis.

### Scripts

Running scripts are provided for the Cellpose models listed above.

Input and output paths should be updated according to the local computing environment.

### Parameter Notes

Cellpose models were mainly run with default parameters to keep the analysis comparable across methods.

Large image files and complete segmentation outputs are not stored in this repository.

### Reference

Official Cellpose repository: https://github.com/MouseLand/cellpose

## 中文

本文件夹用于保存 Cellpose 相关的运行脚本、环境文件和流程记录。

### 包含模型

| 版本 | 模型 |
|---|---|
| Cellpose2 | `cyto2` |
| Cellpose3 | `cyto3` |
| Cellpose3 | `nuclei` |
| Cellpose4 | `cpsam` |

### 环境文件

这里包含三个不同 Cellpose 版本对应的环境文件：

- Cellpose2 环境
- Cellpose3 环境
- Cellpose4 环境

这些文件用于记录分析过程中使用的软件依赖。

### 脚本说明

这里保存了上述 Cellpose 模型对应的运行脚本。

实际运行时，需要根据本地计算环境修改输入和输出路径。

### 参数说明

Cellpose 系列模型主要使用默认参数运行，以保持不同方法之间的分析可比性。

大型图像文件和完整分割输出不保存在该仓库中。
