# Analysis Scripts for Cell Segmentation in Spatial Transcriptomics

This repository is a curated collection of analysis scripts from the part of my undergraduate thesis work that I was responsible for.

This repository only contains the selected tools, selected datasets, and selected analysis scripts that were included in my own component of the work. It is not intended to represent the full project and should not be interpreted as a comprehensive benchmark.

The tools used or involved here include:

- Cellpose
- CellBin2
- Spateo Watershed
- StarDist

To keep the repository lightweight and shareable, raw data, large intermediate results, and complete HTML reports are not uploaded.

More runnable scripts and analysis scripts will be added gradually as the repository is further organized.

## Repository Structure

- `methods/`: method-specific folders for scripts or notes
- `scripts/`: shared analysis scripts and utilities
- `docs/`: documentation for data and parameter notes
- `example_results/`: small example outputs that are safe to share
