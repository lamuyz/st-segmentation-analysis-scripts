# Notes on Parameters

Most tools in this repository currently use default parameters or officially recommended parameter settings.

For the Cellpose series, the models involved may include `cyto2`, `cyto3`, `nuclei`, and `cpsam`.

CellBin2 and Spateo Watershed are expected to follow their built-in workflows unless otherwise noted.

For results related to the `nuclei` model, it is important to distinguish between nucleus segmentation and full-cell region segmentation.

If special modifications, fallback logic, or non-default parameter choices are introduced later, they should be recorded in this document.
